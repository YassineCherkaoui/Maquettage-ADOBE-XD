## Réaliser-un-site-responsive-depuis-sa-maquette

## Contexte du projet

```bash
Pour améliorer sa présence dans le marché une agence de voyage vous a contacté.

votre client représenté par votre formatrice :) vous fournira la maquette pour la page d'accueil et sa charte graphique.

le travail à faire:

Le site web doit contenir les onglets suivantes: (Accueil,About Us, Services, Promo, Galerie, Contact)
Produire un prototype version web/mobile.
Développer des pages web responsive avec HTML/CSS à l’image du prototype réalisé (sans utiliser des frameworks ).
Prévoir un validation du formulaire de la page contact avec Javascript et affichage d'une alerte contenant le message.
Respect de la charte graphique de la maquette
N.B :Vous êtes amenés à être créatif en proposant du contenu adapté (texte corecte /et images).
```
