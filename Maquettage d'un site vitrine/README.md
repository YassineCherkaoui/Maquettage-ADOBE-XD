## Maquettage-d'un-site-vitrine

## Contexte du projet

```bash
Une chaîne hôtelière souhaite créer un site web pour renforcer sa présence web au delà des réseau sociaux. Pour cette raison on vous propose de réaliser le maquettage de son site web vitrine pour les pages d'accueil,les page de présentation des différent (chambres, suites, bungalow...) et page contact avec localisation pour chaque hôtel.

N.B :vous pouvez travailler sur un autre contexte de votre choix
Chercher la différence entre : zoning ,wireframe, maquette,prototype et mockup
Etude de benchmarking entre 3 type d’outils de maquettage et mockup de votre choix.
vous reprenez le design d’un site existant
vous réaliser le zoning et le wireframe de votre site à la main
Le passage entre les pages du siteweb devra être schématisé.
vous réaliser la maquette et le prototype de votre site web sur un outil de maquettage sur lequel on va se mettre d'accord après la présentation
des outils.

La maquette devra être similaire à la maquette choisie
votre site devrait respecter les éléments d'ergonomie et du responsive design

```
